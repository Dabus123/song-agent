# Push to GitHub

After creating a GitHub repository, run these commands:

```bash
# Add your GitHub repository as remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/songcast-xmtp-agent.git

# Push to GitHub
git push -u origin main
```

Or if you prefer SSH:

```bash
git remote add origin git@github.com:YOUR_USERNAME/songcast-xmtp-agent.git
git push -u origin main
```

## Quick Setup

1. Create repo on GitHub: https://github.com/new
2. Name it: `songcast-xmtp-agent`
3. Don't initialize with any files
4. Copy the repository URL
5. Run the commands above with your URL
